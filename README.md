# Twitter Clone

A FullStack Twitter Clone application that replicates core Twitter functionalities, allowing users to post tweets, follow other users, view profiles, and more. Built with a modern tech stack, including Node.js, GraphQL, Prisma, PostgreSQL, Supabase, Redis, Next.js, and AWS.

## Features

- **User Authentication**: Google OAuth integration and JSON Web Token (JWT)-based authentication.
- **Post Tweets**: Users can create, edit, and delete their tweets.
- **Follow System**: Follow and unfollow users to see their tweets on your feed.
- **User Profiles**: View user profiles with details of their tweets, followers, and following.
- **Real-Time Updates**: Real-time updates on the feed using React-Query and GraphQL.
- **Responsive UI**: Optimized for both desktop and mobile viewing.

## Technologies Used

- **Backend**:
  - **Node.js**: Server-side environment.
  - **GraphQL**: API for structured and type-safe data fetching.
  - **Prisma ORM**: Database interaction with PostgreSQL.
  - **Redis**: Caching and managing real-time data.
  - **Supabase**: Open-source Firebase alternative for storage and authentication.

- **Frontend**:
  - **Next.js**: React framework with SSR capabilities.
  - **TailwindCSS**: Utility-first CSS framework.
  - **React-Query**: Data-fetching library for handling server-state.

- **Cloud and Services**:
  - **AWS**: For cloud storage and services.
  - **Supabase**: User authentication and data storage.
  - **GraphQL Codegen**: Automatically generates GraphQL types for TypeScript.
